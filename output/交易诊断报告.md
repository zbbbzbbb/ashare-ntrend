# A股趋势跟踪策略 · 交易诊断报告

> 策略：N型突破 + 行业轮动 + MACD择时  
> 回测区间：2018-01-01 → 2020-12-31  
> 参数：MACD(20,38)，5行业×10仓位，1,500,000 RMB  
> 成本：佣金0.025% + 印花税0.05%(卖) + 滑点0.1% + 冲击0.05%

---

## 一、入场选股全流程

每天收盘后，经过四层过滤决定买什么：

```
沪深300 MACD牛熊  ──→  行业轮动评分  ──→  N型突破选股  ──→  仓位+止损执行
    (第1层)             (第2层)           (第3层)           (第4层)
```

---

### 第1层：大盘择时 —— 今天能不能买？

用沪深300指数的 MACD 判断市场状态。**三个条件必须同时满足**才允许开新仓：

| 条件 | 含义 |
|---|---|
| DIF > DEA | MACD快线在慢线上方 |
| 柱状图绝对值在扩大 | 趋势在加速，而非衰竭 |
| DIF 在上升 | 快线方向向上 |

任意一个不满足 → 当天不开新仓，已持仓可减仓或退出。

2015-2025年间牛市占比仅 25.5%，一半以上的日子不能买入。

---

### 第2层：行业轮动 —— 钱在往哪里流？

从 440 个申万三级行业中，选出前 5 个。每个行业用成分股**等权合成**一条价格曲线，然后按以下维度打分：

| 评分维度 | 权重 | 计算方式 |
|---|---|---|
| MACD 看多 | 25% | DIF > DEA 得 1 分，否则 0 |
| 相对强度排名 | 35% | 20日涨幅在全部行业中的百分位 |
| 成交量放大 | 20% | 当日成交 ÷ 60日均量，封顶 2 倍 |
| N 型形态质量 | 20% | 行业指数自身是否也呈现 N 型突破 |

硬性门槛：MACD 必须看多，否则直接淘汰。达标行业不足 5 个时，取总分最高者凑数。

---

### 第3层：N 型突破选股 —— 具体买哪只？

对入选行业里的每只成分股，检测 N 型形态后综合打分。

#### N 型形态定义

```
价格
 │    ┌── 第一段上升（20天前 → 10天前，涨幅 > 2%）
 │   ╱
 │  ╱  ╲
 │ ╱    ╲___ 回调（10天前 → 5天前，跌幅 < -1%）
 │╱         ╲___ 今天收盘突破前期高点 → 买入信号
─┼────────────────────────────→ 时间
 20天前  10天前  5天前  今天
```

四个条件缺一不可：
- 第一段（T-20 → T-10）：涨幅 > 2%
- 回调（T-10 → T-5）：跌幅 < -1%（必须有回调，不能一直涨）
- 突破：今天收盘 > 近10天最高价
- 突破幅度：任何正幅度的突破都算

#### 股票综合打分

| 评分维度 | 权重 | 说明 |
|---|---|---|
| 个股 MACD 看多 | 25% | 个股 DIF > DEA |
| N 型质量分 | 30% | 第一段涨幅 + 回调深度 + 突破强度的综合 |
| 成交量放大 | 20% | 当日成交量 > 20日均量 |
| 价格在 BBI 上方 | 15% | BBI = (MA3+MA6+MA12+MA24)/4 |
| N 型检测到 | 10% | 只要检测到就给分 |

硬性过滤：MACD 必须看多 **且** 成交量必须放量。合格股票太少时，放宽到只要求 MACD 看多 + 放量（不强制 N 型）。最终取分数最高的前 10 只买入。

---

### 第4层：止损与止盈 —— 什么时候卖？

#### 止损（按入场位置区分）

| 入场位置 | 止损规则 |
|---|---|
| 入场价 **高于** BBI | 连续两天收盘 < BBI → 全平 |
| 入场价 **低于** BBI | 收盘跌破入场价 -3% → 全平 |

> ⚠️ 所有止损在**收盘后**执行，不是盘中。如果当天跳空低开直接跌穿止损线，以收盘价为准。

#### 止盈

持仓盈利达到 +20% → 卖出 50%，剩余 50% 继续持有直到趋势止损触发。

#### 交易频率限制

- 每周最多买入 **2 笔**
- 熊市期间（大盘 MACD 看空）不允许开新仓

## 四、策略诊断

### 胜率为什么只有12%？

| 根因 | 解释 |
|---|---|
| **N型假突破太多** | A股中N型形态出现频率不低，但真正走出第二波的不到两成。形态检测只看价格结构，不看驱动力 |
| **放宽条件引入噪音** | 当合格股票不够时，代码自动放宽到「只要MACD看多+放量」，等同于放弃N型过滤，大量低质量股票进入候选池 |
| **大盘择时太粗糙** | 只用了MACD三个条件，没考虑波动率、成交量、市场宽度。2018年单边下跌市中，MACD偶尔翻多但都是假信号 |
| **行业轮动可能帮倒忙** | 行业指数是用成分股等权合成的，小盘股波动大，行业指数本身也充满噪音。轮动选出的「强势行业」可能只是随机波动 |

### 为什么实际亏损远超-3%止损？

| 原因 | 说明 |
|---|---|
| **止损在收盘执行** | 策略设计是「每日收盘后检查止损」，但A股T+1，买入当天不能卖，次日盘中跳空也无法盘中止损 |
| **跳空低开无法防范** | 兰石重装T+1日跌5.5%、正邦科技T+1日跌5.2%，都是开盘直接跳空，收盘才执行止损，实际亏损远超-3% |
| **涨跌停板限制** | A股涨跌停10%，突破买入后如果遇到利空，直接封跌停，想卖也卖不掉 |

### 关于T+1：代码是合规的

回测引擎中 `process_daily()` **先处理卖出、再处理买入**，当天买入的股票不会在当天被卖出。报告中所有交易最短持仓1天（T+1），没有同一天买卖同一只股票的情况。但A股T+1只能保证隔日卖，不能保证盘中止损——这是现实限制，不是回测bug。

### 为什么趋势走出来的6笔，持仓都要一个月以上？

因为止盈线设在+20%。在A股正常波动下，涨20%通常需要20-40个交易日。而那些1-5天就止损的，根本没给趋势走出来的时间。

## 五、改进方向

| 优先级 | 方向 | 具体措施 |
|---|---|---|
| 🔴 最高 | 降低假突破 | 增加突破当天的**成交量突增要求**（如成交量 > 20日均量的1.5倍，而非仅仅 > 1倍） |
| 🔴 最高 | 降低假突破 | 提高N型质量分阈值，只买入 `pattern_quality > 0.5` 的股票 |
| 🔴 最高 | 防止跳空风险 | 增加**大盘波动率过滤器**，VIX/波动率过高时暂停买入 |
| 🟡 高 | 改善盈亏比 | 把止盈从固定+20%改为**追踪止盈**（如从最高点回撤8%就卖） |
| 🟡 高 | 降低集中爆仓 | 限制单行业最大仓位（目前没有，2020年2月所有仓位同时被击穿） |
| 🟢 中 | 提高胜率 | 去掉「放宽N型要求」的逻辑，宁可空仓也不降低标准 |
| 🟢 中 | 行业轮动 | 改为直接全市场选股（跨行业RS排名），跳过行业合成这一步 |
